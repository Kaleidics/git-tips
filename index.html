CSS: Media-Query Mobile to Desktop

Purpose: A media query for smaller screen size to larger.

How to: Write CSS within the brackets of the media query. When maximized to the breakpoint, apply the styles. 

                                
                                    @media screen and (min-width: 1024px) {
                                        //styles here
                                    }

CSS: Media-Query Desktop to Mobile

Purpose: A media query for large screen size to smaller.

How to: Write CSS within the brackets of the media query. When minimized to the breakpoint, apply the styles. 

                                
                                    @media(max-width:720px) {
                                        //styles here
                                    }

Git: Delete all commit history in Github

Purpose: Deletes all commit history but keep the code in its current state.

How to: Deleting the .git folder may cause problems in your git repository. If you want to delete all your commit history but keep the code in its current state, it is very safe to do it as in the following: 

                                
                                    //1. Checkout
                                    git checkout --orphan new_branch
                                    //2. Add all the files
                                    git add .
                                    //3. Commit the changes
                                    git commit -m "commit message"
                                    //4. Delete the branch
                                    git branch -d master
                                    //5. Rename the current branch to master
                                    git branch -m master
                                    //6. Force update your repository
                                    git push -f origin master

Git: Create branch and switch to it

Purpose: Creates a new branch and switches to it at the same time.

How to: Run git checkout command with the -b switch 

                                
                                    $ git checkout -b my_new_branch
                                    Switched to a new branch "my_new_branch"
